The game in one breath

Both players start with 30 cards — 10🪨 10📄 10✂️ — and 0 Crafticoins. Every turn has two phases. Shop: both players secretly buy at most one thing; purchases are then revealed to both sides. Attack: both players pick a card, simultaneously. Loser's card is destroyed forever; winner's card (and both cards on a tie) go back to hand. Run out of cards and you lose. Survive 100 turns and the bigger hand wins. Everything is public: hands, tiers, coins, purchases, history.

Crafticoins 🪙

Attack with the same shape two turns in a row → +2 Crafticoins (even if the card died). Bombs have no shape — playing one breaks your combo chain. You can't spend a coin the same turn you earn it.

The shop (one purchase per turn, all purchases permanent)

⬆ upgrade2🪙 tier 0→1
1🪙 above		+1 tier on one specific card. Same shape: higher tier wins the clash, equal tiers tie. Different shapes ignore tiers — plain 📄 still eats 🪨+9. Winners keep their tier forever.
💣 bomb5🪙a card that trades: play it and both played cards are destroyed, no winner, no loser. Guaranteed 1-for-1, no prediction needed.
🛒 card7🪙a brand-new tier-0 card of any shape, straight into your hand.

⚠ Danger rounds

Every 5th turn (5, 10, 15…) the clash loser burns one extra random card from their hand — upgraded cards and bombs included. Ties and bomb trades are safe. Plan around the clock.

Illegal token, crash, or blowing the per-phase time limit (default 300 ms) forfeits the game — humans in the Play tab get a retry instead.

Write your bot in four steps

  1. Open bots/my_bot.cpp — it's a working bot with the whole API in comments.
  2. Fill in chooseShop() and chooseAttack().
  3. Hit ⟳ Rebuild bots up top (or rerun the start script / python3 build.py).
  4. Fight it here — or copy the file to bots/your_name.cpp to enter the tournament.

The API you'll actually use

Shop chooseShop(const State& s) override {
  return Shop::none();               // save up (always legal)
  return Shop::upgrade(ROCK, 0);     // one r0 card -> tier 1   (2🪙, then 1🪙)
  return Shop::bomb();               // 5🪙
  return Shop::card(PAPER);          // 7🪙
}

Attack chooseAttack(const State& s) override {
  s.me.count(ROCK)           // cards of a shape    s.me.coins, s.me.bombs
  s.me.cards[ROCK]           // map: tier -> count  s.me.maxTier(ROCK)
  s.opp...                   // everything about them — perfect information
  s.myShopNow, s.oppShopNow  // this turn's revealed purchases!
  s.isSpecialRound()         // danger round? loser burns an extra card
  s.legalAttacks()           // every card you can play
  s.next(mine, theirs)       // full one-turn simulation → State

  return Attack::card(ROCK, 0);      // "r"  — plain rock
  return Attack::card(ROCK, 3);      // "r3" — your tier-3 rock
  return Attack::bombCard();         // "!"  — trade 1-for-1
}

Command line

python3 build.py                    # build engine + every bot (any OS)
python3 build.py --test             # rules unit tests
./build/arena play my_bot gambit    # one game, pretty-printed
./build/arena match my_bot gambit -n 100
./build/arena tournament --games 10
./build/arena play my_bot gambit --json   # replay JSON (what this UI uses)
# Windows: the binary is build\arena.exe — everything else is identical

Deterministic reruns: add --seed N. Your bot's RNG (rps::rng()) is seeded by the arena.

House rules the engine enforces

  • Purchases resolve before attacks — you may play the card you just bought or upgraded.
  • Shop costs are paid immediately; this turn's combo income arrives after the clash.
  • Tiers only matter inside a mirror. Cross-shape, the classic triangle is absolute.
  • Bomb vs bomb: both die. Bombs count as cards for the win condition.
  • Both players eliminated on the same turn → draw. Turn 100 → most total cards wins, equal is a draw.
