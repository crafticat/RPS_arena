The game in one breath

Both players start with 21 cards — 7🪨 7📄 7✂️ — and pick one card each turn, simultaneously. Loser's card is destroyed forever; winner's card (and both cards on a tie) go back to hand. Run out of cards and you lose. Survive 100 turns and the bigger hand wins. Everything is public: both sides see both hands, coins, and history.

Crafticoins 🪙

Play the same base shape two turns in a row → +1 Crafticoin (even if the card died; disguises count as their base; you can't spend a coin the same turn you earn it).

Upgrades

r+1🪙beats the plain version of itself (r+ beats r); normal rules vs everything else; r+ vs r+ ties
r?p1🪙your rock fights as paper this turn; if it survives it returns as a rock; coin combos use the real base shape
r!2🪙bomb: if it loses, the card that beat it is destroyed too; on a win or tie it just returns as a normal card

One upgrade per card per turn. A disguise must be a different shape. Illegal move, crash, or blowing the per-move time limit (default 300 ms) forfeits the game — humans in the Play tab get a retry instead.

Write your bot in four steps

  1. Open bots/my_bot.cpp — it's a working bot with the whole API in comments.
  2. Change choose(const State& s) to return a better Move.
  3. Hit ⟳ Rebuild bots up top (or rerun the start script / python3 build.py).
  4. Fight it here — or copy the file to bots/your_name.cpp to enter the tournament.

The API you'll actually use

Move choose(const State& s) override {
  s.turn                    // 1-based turn number
  s.me.hand[ROCK]           // your cards left (PAPER, SCISSORS too)
  s.me.coins                // your Crafticoins
  s.opp.hand, s.opp.coins   // theirs — perfect information
  s.hasHistory()            // false on turn 1
  s.myLast(), s.oppLast()   // last turn's Moves
  s.legalMoves()            // everything you can afford right now
  s.next(mine, theirs)      // full one-turn simulation → State

  return Move::play(ROCK);          // free
  return Move::plus(PAPER);         // p+   1 coin
  return Move::shift(ROCK, PAPER);  // r?p  1 coin
  return Move::bomb(SCISSORS);      // s!   2 coins
}

Command line

python3 build.py                    # build engine + every bot (any OS)
python3 build.py --test             # rules unit tests
./build/arena play my_bot gambit    # one game, pretty-printed
./build/arena match my_bot gambit -n 100
./build/arena tournament --games 10
./build/arena play my_bot gambit --json   # replay JSON (what this UI uses)
# Windows: the binary is build\arena.exe — everything else is identical

Deterministic reruns: add --seed N. Your bot's RNG (rps::rng()) is seeded by the arena.

House rules the engine enforces

  • Costs are paid when you submit; the coin you earn this turn arrives after.
  • x+ only out-ranks the same effective shape — p still beats r+, and r+ beats p?r.
  • A bomb explodes only on a loss — even bombs and upgraded cards die in the blast.
  • Both players eliminated on the same turn → draw. Turn 100 → most cards wins, equal is a draw.
